# Entendendo como acessar meu projeto pela URL

Todos os projetos da Fabricainfo é estruturado em docker, então é importante estudar os comandos básicos do docker.
Exemplo, quero saber onde está sendo executado o projeto amc trânsito: 

$ cd /projects/godocs-cliente

Desejo saber em qual porta o projeto está configurado:

$ docker compose ps

Esse comando vai retornar os containers do projeto e suas respectivas informações, como porta, nome do container e etc ...

Posterior a isso, atualize a branch que você está para evitar conflitos com os coleguinhas, então execute:

$ git status # Retorna o status de arquivos da raiz do seu projeto
$ git branch # Retrona o nome da branch que seu projeto está
$ git pull origin <nome branch> # Atualiza a branch de acordo com bitbucket

# Importante

Sempre que iniciar uma atividade, fazer ela em uma branch separada (nunca faça na develop ou master).
